Gestione di un Giardino Botanico
Questo progetto consiste in un'applicazione Java che si connette a un database MySQL per gestire i dati di un giardino botanico. L'applicazione offre funzionalità CRUD (Create, Read, Update, Delete) per le categorie di piante e le piante stesse.

Requisiti
Java Development Kit (JDK) versione 8 o successiva
MySQL Server
Librerie JDBC per MySQL (solitamente fornite dal driver MySQL)

Utilizzo
Una volta avviata l'applicazione, sarà possibile:

Aggiungere, visualizzare, modificare ed eliminare categorie di piante utilizzando l'interfaccia grafica.
Inserire manualmente i dati richiesti, come nome della categoria, nome della pianta, nome scientifico [...]
Le operazioni di modifica ed eliminazione richiederanno l'inserimento dell'ID del record corrispondente.
